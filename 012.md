# やる気満々の人

## 使い方

```
studio = AStudio()
shape = RollingPolygon(width=200, N=9)
studio.append(shape)
studio.append(AImage(image='kakedasu_school.png',width=180))
for i in range(50):
    studio.render()
studio.append(shape)

IPython.display.Image(studio.create_anime())
```
